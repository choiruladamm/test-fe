## Development

Here are the steps to run the project locally.

1. Clone the repository.

```sh
git clone https://github.com/choiruladamm/test-fe
```

2. Install the dependencies.

```sh
npm i
```

3. create .env files
```sh
VITE_BASE_URL=https://devfortest.my.id
```

4. Run the project.

```sh
npm run dev
```
